# MENSTRA – Herbal Patch for Period Pain Relief

**Live Demo:** https://samruddhiborle.github.io/menstra/


## Overview
MENSTRA is a web-based marketing landing page developed to present a herbal solution for menstrual pain relief. The project focuses on clarity, usability, and visual consistency, ensuring that users can easily understand the product’s purpose and benefits.

The interface is designed with a user-centric approach, providing a seamless experience across multiple devices while maintaining a structured and aesthetically balanced layout.


## Objectives
- To design a responsive product landing page  
- To present a clear problem–solution narrative  
- To implement basic interactivity using web technologies  


## Features
- Responsive design across devices  
- Clean, product-focused UI  
- Interactive “Buy Now” modal  
- Waitlist form with validation  
- Smooth scrolling  


## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)
